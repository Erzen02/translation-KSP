# translation-KSP

## About

自分が作成したKerbal Space ProgramのMODの日本語訳データを保管するリポジトリです。
ライセンスに従って使用できます。
翻訳データの内容の正確性は保証されません。誤字訂正等はissueへ。

## How to use

- リポジトリ内容をそのままゲームディレクトリ内に展開すれば使えると思います。
- 翻訳とチェックが完了するするごとに各MODのリポジトリにプルリク出す予定なので本家が更新しないときに使ってください。
